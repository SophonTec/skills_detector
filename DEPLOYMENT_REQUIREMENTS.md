# 部署环境要求

## 最低配置

| 组件 | 要求 | 说明 |
|------|------|------|
| 操作系统 | Linux/macOS/Windows | Ubuntu 20.04+, macOS 12+, Windows 10+ |
| CPU | 2 cores | 任意现代 CPU |
| 内存 | 2GB | 推荐 4GB+ |
| 磁盘 | 10GB | 推荐 50GB+ SSD |
| 网络 | 公网 | 需访问外部 API |
| Docker | 20.10+ | 推荐生产环境 |
| Docker Compose | 2.0+ | 推荐生产环境 |

---

## 不需要的组件

❌ **不需要 AI 模型**
❌ **不需要 GPU**
❌ **不需要 AI API Key**
❌ **不需要专用数据库服务器**
❌ **不需要负载均衡器（小规模）**

## 快速检查脚本

运行此脚本验证环境：

```bash
bash deploy/check_environment.sh
```

检查项：
- Docker 版本
- 磁盘空间
- 内存大小
- 网络连接
- 端口可用性
